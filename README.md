## About

**Countries - Capitals Game** is a console game.
The player is shown either a country or a capital and must type the correct counterpart.
The game keeps track of the score until all pairs are completed.

Features:

- Choose to guess either capitals or countries
- Randomized pairs for each game session
- Internal game logic implemented following **Test-Driven Development (TDD)** methodology

## Documentation

- Input: type the corresponding name when prompted with "Country" or "Capital"
- Pairs: loaded from text files EuropeCapitals.txt and EuropeCountries.txt. They need to be located next to .exe
- Score: the program tracks correct answers
- Game ends after all pairs are completed
- Wrong answer: the program shows the correct answer in the format Country - Capital

## About

**Countries - Capitals Game** — это консольная игра на C++.
Игроку показывается страна или столица, и нужно ввести правильный ответ. Игра ведёт подсчёт очков, пока не будут пройдены все пары.

Особенности:

- Возможность выбирать угадывать столицы или страны
- Случайное перемешивание пар для каждой игры
- Внутренняя логика игры реализована согласно методологии **Test-Driven Development (TDD)**

## Documentation

- Ввод: при запросе "Страна" или "Столица" введите соответствующее название
- Пары: загружаются из текстовых файлов EuropeCapitals.txt и EuropeCountries.txt. Они должны находится рядом с .exe
- Счёт: программа ведёт подсчёт правильных ответов
- Игра завершается после прохождения всех пар
- Неверный ответ: программа показывает правильный ответ в формате Страна - Столица
